# TAREA PRACTICA 1 PERIODO II

Repositorio de apps hechas en Swift. Tarea en parejas.

## Tecnologías usadas

* Hypervisor: **VMware Workstation Pro**
* Sistema Operativo: **macOS Big Sur**
* IDE: **Xcode**
* Lenguaje de programación: **Swift**

## Probar las aplicaciones

Clonar el repositorio:

```bash
git clone https://github.com/JoseDev155/swift-xcode-practice.git
cd swift-xcode-practce/
```

Para verificar que fueron hechas por el alumno, ejecutar:

```bash
git log
```
